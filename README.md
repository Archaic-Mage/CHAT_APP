# Getting Started

Download the project contents and run the following command in both the client and server directory

### `npm install`

Install the packages and its dependencies needed to run the program.


## How to Start

Start both the client and server by running the following command in respective directories.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
